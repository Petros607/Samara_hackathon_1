# Samara_hackathon
Веб-сервис, который принимает ссылку на запись лекции с платформы BigBlueButton (BBB), обрабатывает её с помощью ИИ-технологий и формирует структурированный конспект в формате PDF. В результате пользователи получают готовый конспект без необходимости вручную записывать ключевые моменты лекции.
